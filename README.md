# The Low Level Journey

Aprendiendo programación de bajo nivel con google antigravity.

Antigravity permite una especie de persistencia llamada "Workspace Rules". Estas reglas viven en la carpeta `.agent/rules/` y son aplicadas automáticamente por el Agente en este espacio de trabajo.

Para nuevos entornos:
1. Asegúrate de que la carpeta `.agent/rules/` existe.
2. El archivo `instructions.md` dentro de esa carpeta guiará al Agente.

## Reglas de Oro del Proyecto:
- **Objetivo:** Crear un sistema operativo desde cero.
- **Documentación:** Todo en `docs/`, cronológico y atractivo.
